# DeepEPI

This repository contains the scripts, data, and trained models for DeepEPI. DeepEPI is a deep learning model to predict enhancer-promoter interactions by employing features from large genomic contexts.

# Requirements

* numpy
* scikit-learn
* PyTorch>=1.6.0
